# PKN KELAS 2 SD

KISI-KISI ASAT PKN KELAS 2

No,Materi,Indikator Soal
1,Musyawarah,Peserta didik dapat memahami musyawarah sesuai nilai Pancasila.   
2,Lambang Pancasila,Peserta didik dapat mengenal semboyan pada lambang Pancasila.   
3,Nilai Pancasila,Peserta didik dapat mengenal perilaku sesuai nilai Pancasila.   
4,Manfaat Pancasila,Peserta didik dapat memahami manfaat penerapan Pancasila.   
5,Kewajiban di rumah,Peserta didik dapat memahami kewajiban anak di rumah.   
6,Peran ayah,Peserta didik dapat memahami tugas ayah dalam keluarga.   
7,Hak dan kewajiban,Peserta didik dapat memahami manfaat hak dan kewajiban.   
8,Kewajiban di sekolah,Peserta didik dapat memahami kewajiban menjaga lingkungan sekolah.   
9,Hidup harmonis,Peserta didik dapat memahami manfaat mematuhi hak dan kewajiban.   
10,Keberagaman,Peserta didik dapat memahami manfaat menghargai keberagaman.   
11,Persatuan,Peserta didik dapat memahami pengertian persatuan.   
12,Peran sekolah,Peserta didik dapat memahami peran sekolah dalam persatuan.   
13,Toleransi,Peserta didik dapat memahami upaya menciptakan persatuan.   
14,Menjaga lingkungan sekolah,Peserta didik dapat memahami cara menjaga lingkungan sekolah.   
15,Gotong royong,Peserta didik dapat memahami manfaat gotong royong.   
16,Persatuan,Peserta didik dapat memahami arti persatuan.   
17,Kerja kelompok,Peserta didik dapat memahami sikap kerja sama.   
18,Manfaat persatuan,Peserta didik dapat memahami manfaat menjaga persatuan.   
19,Merawat lingkungan,Peserta didik dapat memahami sikap menjaga lingkungan sekolah.   
20,Tujuan persatuan,Peserta didik dapat memahami tujuan persatuan.
